### Setup
```bash
yarn add express body-parser cross-env

# support es6/7 in Node
yarn add -D babel-cli babel-plugin-transform-object-rest-spread babel-preset-env

# listening to any changes in Node.js files
yarn add -D nodemon

# eslint setup
yarn add -D eslint eslint-config-equimper prettier eslint-config-prettier
```

### How to run
```bash
mongod # running database
yarn dev # running dev env
yarn prettier # make the code prettier
mongo # to access to mongodb shell
```
